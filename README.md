# Blog Sphere

Bem-vindo ao Blog Sphere! Este é um projeto para compartilhar ideias, pensamentos e conhecimentos através de postagens de blog. Nosso objetivo é inspirar e conectar os leitores com uma diversidade de tópicos, incluindo tecnologia, criatividade, estilo de vida, produtividade e muito mais.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Instalação

Siga os passos abaixo para rodar o blog localmente:

1. Clone o repositório:
    ```bash
    git clone https://github.com/seu-usuario/blog-sphere.git
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd blog-sphere
    ```
3. Instale as dependências:
    ```bash
    npm install
    ```
4. Inicie o servidor:
    ```bash
    npm start
    ```

## Estrutura de Arquivos

A estrutura de arquivos do projeto é a seguinte:

```
.gitignore
index.html
LICENSE
public/
    assets/
        fonts/
            fonts.css
        images/
README.md
src/
    artigos-post/
    css/
        artigos.css
        base.css
        footer.css
        header.css
        main.css
        style.css
    scripts/
        categorias.js
        main.js
        menuBtn.js
```

- `index.html`: Página principal do blog.
- `public/`: Contém os ativos públicos como fontes e imagens.
- `src/`: Contém os arquivos de código fonte, incluindo artigos, estilos CSS e scripts JavaScript.

## Como Contribuir

Contribuições são bem-vindas! Sinta-se à vontade para abrir issues e enviar pull requests. Caso queira enviar sugestões ou novos artigos, siga o processo abaixo:

1. Faça um fork do repositório.
2. Crie uma nova branch para sua contribuição:
    ```bash
    git checkout -b minha-contribuicao
    ```
3. Faça as alterações desejadas e commit:
    ```bash
    git commit -m "Minha contribuição"
    ```
4. Envie para o seu fork:
    ```bash
    git push origin minha-contribuicao
    ```
5. Abra um pull request no repositório original.
